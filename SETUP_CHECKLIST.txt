## ğŸ’¾ State Management

**Redux Store Structure:**
```javascript
{
  cart: {
    items: [
      { id, title, price, image, quantity, totalPrice }
    ],
    totalPrice: number,
    totalQuantity: number
  }
}
```

**Available Actions:**
- `addToCart` - Add product
- `removeFromCart` - Remove product
- `increaseQuantity` - Increase item qty
- `decreaseQuantity` - Decrease item qty
- `clearCart` - Empty cart
